# TestWeave

**Умный генератор Jest-тестов для JavaScript-проектов**

TestWeave автоматически создаёт заготовки тестовых файлов для ваших JS-модулей — быстро и удобно.

## Возможности

### 1. Запуск из VS Code (рекомендуемый способ)

- Нажмите правой кнопкой мыши на папку в Explorer - **Генерация js-тестов** - **Запуск**
- При первом запуске в проекте расширение попросит выбрать папки
- При следующих запусках — работает **мгновенно**, без лишних вопросов

### 2. Запуск через CLI (командную строку)

```bash
node generate-tests.js ./src --output ./tests
```

## Что умеет TestWeave

### Автоматическое создание тестов

#### Без расширения вам приходилось:

- вручную создавать файл something.test.js
- писать describe, test, require для каждого модуля
- повторять это для десятков файлов

#### TestWeave делает это за вас:

- Находит все .js файлы в папке
- Определяет, какие функции экспортируются
- Создаёт готовые тесты с проверками toBeDefined() и test.todo()
- Сохраняет структуру папок

### Поддержка разных типов модулей

- Обычные Node.js модули (CommonJS)
- Модули с несколькими экспортируемыми функциями
- Браузерные скрипты (IIFE) — при включении флага `--iife`

### Умные настройки

- Запоминает параметры для каждого проекта отдельно
- Поддерживает перезапись файлов (--overwrite)
- Может пропускать или обрабатывать index.js

## Установка

- Установите расширение TestWeave из Visual Studio Code Marketplace
- Откройте папку проекта
- Нажмите правой кнопкой на нужной папке - Генерация js-тестов - Запуск

Готово! Тесты будут созданы автоматически.
